Kiosk Portal

A kiosk website I wrote for the North & South Regions

Overview

The kiosk website is a website I wrote out in blazor to display company related links. The site itself is hosted in azure. Filtering is done via IP address.

Architecture & Tech Stack

  • Programming Language: Blazor (C# + HTML + CSS)
  • Hosting: Azure

Key Features


Unified Dashboard

A unified spot for either region to access relevant sites


Simple in design

The way the site is written it can be picked up and worked on by anyone


Security

By allowing for ip filtering there's no worry of a outside threat
