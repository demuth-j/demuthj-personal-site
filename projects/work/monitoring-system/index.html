Global Monitoring System

A global infrastructure monitoring system I rolled out for the company

Overview

A repeated issue at the company was servers or services stalling out and certificates expiring. I implemented a global monitoring system to solve these issues and give the IT team more insight into what is happening. I built the entire system myself.

Architecture & Tech Stack

  • Programming Language: Terraform & YAML
  • Data Collection: Prometheus for collecting and thanos for storage
  • Data Display: Grafana
  • Hosting: Azure Kubernetes Cluster for hosting and Azure blob for long term storage

Key Features


Global Monitoring of IT infrastructure

This system was deployed per region allowing each team to have their own monitoring regardless of if a region is offline


Security First Implementation

This system was implemented with HTTPS and SSO as well as only necessary exposing down for services and ports
