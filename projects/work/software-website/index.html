Approved Software Website

A website I wrote for showing what software is approved to end users

Overview

The software website is a website I wrote out in blazor that end users could use to find approved software. The website features admins and users which is determined by entra security groups.

Architecture & Tech Stack

  • Programming Language: Blazor (C# + HTML + CSS) and a SQl backend
  • Hosting: Azure

Key Features


Unified Place to Find Software

A unified spot for any employee to find what software they are allowed to download


Simple in design

The site is simple enough and responsive enough that even the most senior employee could navigate it


Security

By implementing rbac employees only access what is relevant for their role
