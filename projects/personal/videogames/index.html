Videogames & Game Jams

A passion for games that has turned into me learning game development

Overview

Orginally I entered my first game jam to see simply if I could make a game in a short amount of time and now it has become one of my favorite hobbies. A game jam is a event in which the developer has a set amount of time to make a game while following a specific theme. The theme itself can be interpreted in a variety of ways which allows for creative games to be made.

Architecture & Tech Stack

  • Game Engine: Unity
  • Programming Language: C#

The game involves the player navigating a dark town looking for key items while tryin to avoid roaming enemies. I made this game for 72 hour game jam and tested my ability to make hand made maps and tweak gameplay numbers such as player speed as well as enemy line of sight.

The Inventory Specialist - Winter MelonJam 2025 GameJam Submission

This was a horror game made in 72 hours around the the theme of mask. This game involved me learning to program ai, working in a 3d environment, audio design, and animations. The game itself involves the player retrieving items in a very dark basement with a roaming monster and deadly gas. Without the gas mask the player quickly dies. This was made in the unity engine and used publicly available assets.
