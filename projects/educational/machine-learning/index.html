Overview

The page features a list and brief description of various machine learning projects I worked on.

Architecture & Tech Stack

  • Programming Language: Python
  • Tools: Jupyter Notebook
  • Libraries: Pandas, Numpy, XGboost, Sklearn

Projects


Making YouTube Titles More Clickbait

The goal of this project was to fine tune ChatGPT utilizing reinforcement learning through human feedback to create more appealing youtube video titles based off a dataset of top youtube video titles.


American History GPT

This project involed making a GPT model from scratch and training it on american history books with the goal to act as a history teacher.


Food Image GAN

This project involved me utilizing various fast food images (10 labels, 15k images) to train a adversarial GAN to generate images of food items.


Cifar 10 CNN

For this project I had to create a CNN that could classify images from the CIFAR 10 dataset. My CNN was the best in the class with a overall accuracy, precision, and recall of 0.75
