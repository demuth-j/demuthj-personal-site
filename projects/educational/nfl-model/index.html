NFL Game Predictions Utilizing Game and Non Game Related Features

A semester long research project that involved utilizing existing research to train a custom ai model for game prediction

Overview

This project involved using machine learning to predict the outcome of the 2016 to 2022 NFL seasons using both game features (average yards, average INTs, etc.) and non game features (stadium capacity, temperature, etc.) with the goal of testing the impact of non game features on a NFL game. My model's average accuracy was 64.55% and my findings concluded that there is minimal impact of non game features on the result of NFL games.

Architecture & Tech Stack

  • Programming Language: Python
  • Tools: Jupyter Notebook & Excel
  • Libraries: Pandas, Numpy, XGboost, Sklearn

Overall Thoughts

Overall this project was fun to work on and reinforced my machine learning skills. I learned alot about working with data cleanup and parameter optimization. I continue to work on other sports prediction models to this day.
