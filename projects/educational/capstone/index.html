Overview

The goal of this project was to provide the framework required for a machine learning model to communicate with patients who are in the early stages of dementia and navigate the conversation through a series of questions while having the conversation feel natural to the patient. While generalized chat bots exist, there are no chat bots created for those with dementia. The completion of this project includes a webpage featuring an avatar and computer-generated voice as well as an exploration into which model and prompt will be ideal to use in the future stages of this research.

Architecture & Tech Stack

  • Programming Languages: PHP and JavaScript
  • Avatar: Generated using MidJourney
  • Voice & Video : D-ID
  • LLM: ChatGPT

Overall Thoughts

Overall this paper helped give me solid skills both in machine learning and with project management. I am glad that I was able to have a impact on both the machine learning and medical community. I hope to continue to make impacts in these fields.
